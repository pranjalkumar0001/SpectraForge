# SpectraForge

A modular signal-processing and spectral-analysis framework written in C++.

## Overview

SpectraForge is a project focused on implementing signal-processing algorithms from first principles, including Fourier transforms, spectral analysis, and digital filtering.

## Planned Features

### Signal Generation
- Signal class
- Sine wave generation
- Multi-tone signal generation
- Noise generation

### Spectral Transforms
- DFT
- FFT
- Magnitude spectrum

### Spectrum Analysis
- Dominant frequency detection
- Peak detection

### Digital Filters
- Low-pass filters
- High-pass filters
- Butterworth filters

### Data I/O
- CSV import/export

### Future Extensions
- Audio analysis
- EEG analysis
